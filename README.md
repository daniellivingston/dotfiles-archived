# `dotfiles`

## Setting Up

In `~/.zsrc`, the only line should be:

    source ~/path/to/dotfiles/shell/.zshrc

Or,

    ln -s ~/path/to/dotfiles/shell/.zshrc ~/.zshrc

## Dependencies

```sh
$ brew install pure \
        zsh-autosuggestions \
        zsh-syntax-highlighting
```
